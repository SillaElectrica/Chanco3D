# Chanco3D

Este repositorio contiene el código fuente de la plataforma web oficial de Chanco3D, un proyecto académico y emprendedor enfocado en la impresión 3D personalizada en resina.

La plataforma web es el componente central del proyecto, diseñada para optimizar la presentación de servicios, facilitar la interacción con los clientes y centralizar la gestión de pedidos, clientes y materiales.
